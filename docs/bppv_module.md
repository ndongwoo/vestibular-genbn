# BPPV worked example

The BPPV module uses positional history and positional nystagmus patterns. The active disease
nodes include posterior-canal BPPV, horizontal-canal BPPV, subjective/possible BPPV, and a
central positional mimic node.

Typical posterior-canal BPPV is supported by brief positional history and torsional upbeating
nystagmus on Dix-Hallpike or a compatible side-lying pattern.
